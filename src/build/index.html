Hyhyhy

@_czyzewski

Single, global Hyhyhy:

$ pip install hyhyhy

Configuration over code

[head]
title = Example presentation by hyhyhy!
description = A little words...

[core]
build = build/index.html

[sections]
5 = data-bespoke-state="green" class="light"

Built-in create task to get you started

$ hyhyhy create

Kitchen sink

  • build
  • create
  • status
  • help

Basic structure

|-- assets
|   |-- index.jinja
|   |-- scripts
|   |   |-- main.js
|   |-- styles
|   |   |-- main.css
|-- sections
|   |-- 1.html
|   |-- 2.md
|   |-- ...
|   |-- n.html
|-- default.cfg

$ hyhyhy build
